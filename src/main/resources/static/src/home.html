Technology :

This a simple project is using

  • AngularJS for databinding and presentation
  • Bootstrap & Bootflat for styling and templating
  • Firebase for storing data

Developement environment :

  • SublimeText3 for writing code (duh)
  • NodeJS for local test
  • Firebase hosting for online hosting

Display anime list

This project retrieve a list of anime stored in an online Firebase. To initialize the data in Firebase, we used a report from Anime News Network. Firebase allow you to import JSON object into the database. But, Anime News Network gives you an report in XML Format. So we made a JAVA project to convert XMl to JSON. This project will be available on GITHUB later.
